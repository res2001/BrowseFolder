BrowseFolder (C) res2001, 2016

Отображает стандартное диалоговое окно выбора папки. 
В случае успешного выбора папки полный путь к выбранной папке выводится в stdout и программа завершается с кодом возврата 0.
Если пользователь нажал на кнопку "Отмена" - ничего не выводится и программа завершается с кодом возврата 14.
Программа написана на C++ с использованием "чистого" WinAPI, без стандартной библиотеки.

BrpwseFolder.exe [/Help] [/Description:<описание>] [/Title:<заголовок>] [/Flag:<флаг>]
                 [/Block:<окно блокировки>] [/Center:<окно для центрирования>] [/CurPath:<папка>] 
                 [/RootPath:<корневая папка>]

Параметры:

/Help					- экран помощи.
/Description:<описание>			- описание диалога, выводится над деревом папок. По умолчанию - пусто.
/Title:<заголовок>			- заголовок окна выбора папок. По умолчанию - "Обзор папок"
/Block:<окно блокировки>		- указывает какое окно будет блокироваться во время отображения окна выбора папок.
					  По умолчанию - window.
	<окно блокировки>		- возможные значения:
						none	- не блокировать окна.
						window	- блокировать окно консоли, из которой запущена BrowseFolder.exe.
						<заголовок окна> - блокируется окно с заданным заголовком.
/Center:<окно для центрирования>	- указывает окно, относительно которого будет центрироваться окно диалога выбора папок.
					  По умолчанию - window.
	<окно для центрирования>	- возможные значения:
						none	- не центрировать. Окно выбора папок отображается в координатах заданных Windows.
						window	- центрировать относительно окна консоли, из которой запущена BrowseFolder.exe.
						screen	- центрировать относительно окна рабочего стола ОС
						<заголовок окна> - центрировать относительно окна с заданным заголовком.
/CurPath:<папка>			- путь к папке, которая будет выбрана при инициализации диалога.
					  По умолчанию - "Мой компьютер".
/RootPath:<корневая папка>		- папка, выше которой будет не возможно поднятся при выборе папок (корневая папка). По умолчанию - пусто.
	<корневая папка>		- возможные значения:
						desktop	- %USERPROFILE%\Desktop
						computer - "Мой компьютер"
						favorites - %USERPROFILE%\Favorites
						appdata	- %USERPROFILE%\AppData\Roaming
						localappdata - %USERPROFILE%\AppData\Local
						documents - %USERPROFILE%\Documents
						music	- %USERPROFILE%\Music
						pictures - %USERPROFILE%\Pictures
						video	- %USERPROFILE%\Videos
						network - "Сетевое окружение"
						profile	- %USERPROFILE%
						programfiles - %ProgramFiles%
						windows	- %SystemRoot%
						system	- %SystemRoot%\System32
						<папка>	- любая папка файловой системы
/Flag:<флаг>				- стандартные флаги диалога выбора папок.
					  По умолчанию - BIF_RETURNONLYFSDIRS | BIF_USENEWUI | BIF_NONEWFOLDERBUTTON.
					  Подробное описание флагов смотри (поле ulFlag): 
						https://msdn.microsoft.com/ru-ru/library/windows/desktop/bb773205%28v=vs.85%29.aspx
	<флаг>				- возможные значения (в скобках указаны десятичные значения флага):
						BIF_RETURNONLYFSDIRS (1)	- выводятся только папки файловой системы.
						BIF_STATUSTEXT (4)		- отображается поле статуса. Не совместимо с BIF_NEWDIALOGSTYLE.
						BIF_EDITBOX (16)		- отображается поле ввода.
						BIF_NEWDIALOGSTYLE (64)		- используется новый пользовательский интерфейс диалога (изменение размеров окна диалога, контекстное меню файлов и папок и т.п.)
						BIF_USENEWUI (80)		- BIF_EDITBOX + BIF_NEWDIALOGSTYLE
						BIF_NONEWFOLDERBUTTON (512)	- не показывать кнопку "Новая папка".
						BIF_BROWSEINCLUDEFILES (16384)	- отображаются так же и файлы.
						BIF_SHAREABLE (32768)		- отображаются общие папки удаленных компьютеров. Должен быть установлен и флаг BIF_NEWDIALOGSTYLE.
					  Так же доступны флаги: BIF_DONTGOBELOWDOMAIN (2), BIF_RETURNFSANCESTORS (8), BIF_VALIDATE (32), 
					  BIF_BROWSEINCLUDEURLS (128), BIF_UAHINT (256), BIF_NOTRANSLATETARGETS (1024), 
					  BIF_BROWSEFORCOMPUTER (4096), BIF_BROWSEFORPRINTER (8192), BIF_BROWSEFILEJUNCTIONS (65536). 
					  Описание этих флагов смотрите по ссылке, приведенной выше.
					  Указываемое в параметре /Flag значение является суммой десятичных значений флагов. 
					  Например, значение ипользуемой по умолчанию комбинации флагов:
						BIF_RETURNONLYFSDIRS | BIF_USENEWUI | BIF_NONEWFOLDERBUTTON
					  равно 1 + 16 + 64 + 512 = 593. Для указания этого значения нужно использовать следующую запись параметра:
						/Flag:593

Коды ошибок:
0 - успешное завершение программы. В stdout выводится полный путь к выбранной папке.
1 - не возможно получить параметры коммандной строки.
2 - не возможно создать объект BrowseFolderDialog.
3-8 - не корректные значения параметров.
10 - не верный параметр.
12 - ошибка OleInitialize.
14 - пользователь отказался от выбора папки.
15 - ошибка преобразования выбранной папки.
Коды ошибок сопровождаются соответствующими текстовыми сообщениями, которые выводятся в stderr.
