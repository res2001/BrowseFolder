========================================================================
    КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Обзор проекта BrowseFolder
========================================================================

Показывает стандартное окно выбора папок и выдает в стандартный выходной поток полный путь к выбранному файлу.
При этом окно центрируется относительно текущего консольного окна и блокирует консольное окно.
