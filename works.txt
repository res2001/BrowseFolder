Планирую с помощью параметров задавать:
1.хэндл окна родителя (оно будет блокироваться), можно будет задать 0, чтоб ничего не блокировалось, или блокировка окна консоли.
2.Текст в окне выбора файлов.
3.Параметры для SHBrowseForFolder, такие же, которые используются в версии JS, чтоб можно было управлять содержимым окна.
4.Каталог по умолчанию, который сразу будет выбран в окне.
5.Заголовок окна выбора файлов.
6.Отключение центрирования окна выбора файлов относительно родительского - поведение по умолчанию для SHBrowseForFolder. На самом деле поведение по умолчанию - центрирование относительно рабочей области desktopа.
7.Задание root каталога (есть такая возможность в SHBrowseForFolder - это каталог выше которого нельзя будет подняться по дереву каталогов.
